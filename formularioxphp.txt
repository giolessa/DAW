Para explicar de forma simples como funciona um formulário e como o PHP recebe esses dados, vou usar um código simples para explicar:

<?php
// O PHP processa os dados
// Aqui é verificado se o formulário foi enviado via POST
// Isso evita que o código rode antes de clicar no botão
if ($_SERVER["REQUEST_METHOD"] == "POST") {
    
    // Capturando os dados das superglobais
    $nome = htmlspecialchars($_POST['usuario']);
    // Esse $_POST['usuario'] pega o valor do campo <input 
    // name="usuario">
    $email = htmlspecialchars($_POST['email']);
    // o htmlspecialchars() basicamente protege o sisetema
    // caso alguem queira colocar um codifo malicioso, ele bloqueia
    
    //verifica se o nome e o email NÃO estão vazios
    //se não estiverem vazios, mostra uma mensagem de sucesso
    if (!empty($nome) && !empty($email)) {
        echo "<div style='background: #d4edda; color: #155724; padding: 10px; margin-bottom: 20px;'>";
        //<strong> é pra colocar em negrito e <br> para dar quebra 
        //de linha
        echo "<strong>Dados Recebidos com Sucesso!</strong><br>";
        //mostra os dados que o usuario difitou
        echo "Nome: $nome <br> E-mail: $email";
        echo "</div>";
    } else {
       //se nao mostra uma mensagem de erro
        echo "<p style='color: red;'>Por favor, preencha todos os campos!</p>";
    }
}
?>

<!DOCTYPE html>
<html lang="pt-br">
<head>
    <meta charset="UTF-8">
    <title>Estudo de Formulários PHP</title>
</head>
<body>

    <h2>Formulário de Cadastro</h2>
    
    <form action="cadastro.php" method="POST">
    //action="cadastro.php" manda os dados pra esse arquivo
    //method="POST" usa o metodo POST
        <div>
            <label>Nome:</label><br>
            <input type="text" name="usuario" required>
        </div>
        <br>
        <div>
            <label>E-mail:</label><br>
            <input type="email" name="email" required>
        </div>
        <br>
        <button type="submit">Enviar Dados</button>
    </form>

</body>
</html>

Passo a passo:

Página abre → mostra formulário
Você digita nome e email
Clica em Enviar
O navegador manda os dados (POST)
PHP recebe com $_POST
PHP verifica:
se vazio → erro
se preenchido → sucesso
Mostra mensagem
